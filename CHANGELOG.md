# Changelog - Windows XP Profile App

## [Unreleased] - 2025-11-29

### Added
- **Realistic Windows XP Theme**:
  - Authentic "Luna" blue theme colors and gradients.
  - 3D window control buttons (Minimize, Maximize, Close) with hover/active states.
  - Correct title bar gradients (7-stop blue).
  - Taskbar buttons with bevel/depressed effects.
  - Radial gradient desktop background (XP Blue).
- **CRT Effect**: Global scanline overlay for retro aesthetics.
- **Shutdown Functionality**:
  - "Turn Off Computer" button in Start Menu and Login Screen now triggers a shutdown state.
  - Black screen with "It is now safe to turn off your computer" message.
- **Welcome Notification**: Balloon notification on desktop load.
- **Log Off**: Functional "Log Off" button returning to Login Screen.

### Fixed
- **Start Menu**:
  - Fixed overlapping icons on the right side.
  - Replaced broken User icon with a placeholder.
  - Removed non-functional/broken apps (Paint, Calculator, Minesweeper, etc.).
- **Boot & Login Screens**:
  - Replaced generic logos with realistic "wavy" Windows flag and accurate typography.
  - Fixed "Turn Off" button on Login Screen.
- **Desktop**:
  - Removed Recycle Bin as requested.
  - Cleaned up broken app icons.

### Changed
- **Fonts**: Enforced `Tahoma` as the primary font for pixel-perfect XP look.
- **Scrollbars**: Custom styled scrollbars to match the theme.
